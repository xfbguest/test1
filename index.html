How to read the statistics...

The table shows the classification metrics over the labeled Untrained Views
  • Found The total number of Views classified with the corresponding tag, including trained images.
  • Train The number of views trained/total number of Views labeled.
  • n/c Not classified (score < threshold)
  • Recall Percentage of untrained labeled Views that are classified correctly.
  • Precision Percentage of classified untrained labeled Views that are correct.
  • F-Score Geometrical average of Recall/Precision.

The confusion matrix illustrates the separation between the classes. The area of each circle is proportional to the number of Views.The rightmost column shows the images not classified (if any).
